## 策略说明

### 01 — 铜期货双均线趋势跟踪策略

**选择铜的理由：**
铜（"铜博士"）是全球宏观经济的晴雨表，沪铜主力合约流动性居 SHFE 前列，
趋势性强、波动适中，双均线策略天然契合铜的行情特征。

**策略逻辑：**
- 快线（MA20）上穿慢线（MA60）→ 金叉开多
- 快线（MA20）下穿慢线（MA60）→ 死叉开空
- 止损：入场价 ± 2×ATR(14)
- 使用 `TargetPosTask` 自动管理仓位

---

### 02 — 豆粕期货布林带均值回归策略

**选择豆粕的理由：**
豆粕是 DCE 成交量最大的农产品期货，价格受季节性因素影响显著，
非趋势期（约占 60%~70%）呈典型宽幅震荡，均值回归策略高度适配。

**策略逻辑：**
- 价格跌破布林带下轨 + RSI < 35 → 超卖开多
- 价格突破布林带上轨 + RSI > 65 → 超买开空
- 止盈：价格回归中轨附近（±10% 带宽）
- 止损：入场价 ± 1.5×ATR(14)
